### Login User com context api do React

### Projeto
<p>Login de user usando o context api do react, esse hook ajuda a fornecer uma forma de compartilhar dados, entre todos os componentes da mesma árvore de componentes, sem precisar passar explicitamente props entre cada nível.</p>
<p>É indicado para compartilhar dados que podem ser considerado "globais".</p>

### Techs
- ReactJS
- Context API
- Styled Components
- TypeScript
